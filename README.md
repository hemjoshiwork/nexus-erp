# 🏢 Nexus ERP — AI-Powered Business System

A full-stack ERP system enhanced with an AI assistant that enables natural language interaction with business data.

---

## 🚀 Features

* 📦 Inventory management
* 💰 Billing & POS system
* 📊 Sales analytics
* 🔐 Authentication system
* 🧠 AI chatbot for business insights

---

## 🧠 AI Capabilities

Ask questions like:

* “What are my low stock items?”
* “Total sales today?”
* “Which category has highest stock?”

The system processes real ERP data and returns accurate answers.

---

## ⚙️ Tech Stack

* Frontend: Next.js / React
* Backend: FastAPI
* Database: SQLite / MySQL
* AI: LLM API integration
